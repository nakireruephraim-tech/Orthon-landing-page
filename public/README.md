# Public Assets Directory

This directory contains static assets for the Orthon.AI landing page.

## Directory Structure

- `/images` - Logo, icons, and images
- `/fonts` - Custom fonts (if not using Google Fonts)
- `/icons` - Favicon and app icons

## Usage

Files in this directory are served from the root path `/`. For example:
- `public/logo.svg` is accessible at `/logo.svg`
- `public/images/hero.png` is accessible at `/images/hero.png`
